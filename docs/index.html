Twisted Nail

One-Call Construction Trucking & Wholesale Aggregate Supply

This is a Model Context Protocol (MCP) server for Twisted Nail — a small, standalone service that lets AI assistants (Claude, ChatGPT, Cursor, and any MCP-compatible agent) query Twisted Nail's services, aggregate materials, Central Texas service area, and FAQs directly. The result: accurate, first-party answers and a quote hand-off, instead of an AI guessing.

Zero-touch to the website. This runs as its own service on its own address. It does not install into WordPress and does not modify twistednail.com in any way — the site is untouched whether this is on, off, or never built.
